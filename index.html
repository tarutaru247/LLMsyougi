APIキー設定

・「ブラウザに保存」は localStorage に保存し、タブを閉じても残ります（共有PCでは非推奨）。
・「一時的に保存」は sessionStorage にのみ保存し、このタブを閉じると消えます。
・通信は必ず HTTPS で行ってください。
・設定のエクスポートを使うとキーも含まれます。共有前に削除してください。

入力したキーは選択した保存方法でのみ保持されます。不要になったら設定画面で空にして保存するか、ブラウザのストレージを削除してください。

使い方

対局前の準備

  • 対局するには API キーが必要です。API設定から OpenAI / Gemini の API キーを設定してください。Gemini 無料枠を除き、従量課金となります。
  • 現在使用できるモデル：GPT-5.1 / Gemini 2.5 Flash / Gemini 3 Pro Preview

対局者選択

  • 人間 vs 人間：局面編集や動作確認用。
  • 人間 vs 生成AI：人間が先手、AI が後手。後手モデルを対局したい AI に設定。人間が 1 手目を指すと AI が動作開始。
  • 生成AI vs 生成AI：AI 同士を眺めるモード。先手モデルと後手モデルを設定して「対局開始」。泥仕合になりやすく、放置すると API を使い続けるので注意。

モード選択

AI が手を選ぶ方式は 2 種類です。盤面（画像と文字）、棋譜、直前の手などをプロンプトに含めて次の手を考えさせます。

  • 選択式：合法手リストを渡し、その中から 1 手を選択。反則が出にくく安定。
  • 生成式：合法手リストを渡さず自立思考。反則・無効手の可能性あり。使うときは高性能モデル推奨。

AI が無効な手を指した場合は「再生成」を試してください。繰り返す場合は一手戻して別の手を指してください。不具合が発生する場合があります。
