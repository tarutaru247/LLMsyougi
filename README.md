# LLMと将棋を試すデモ

このアプリはブラウザ上で動作し、利用者が用意したAPIキーを使ってLLMと対局するシンプルなデモです。

## APIキー取り扱いの注意
- キーはブラウザの localStorage に平文で保存されます。共有PC・共有ブラウザでは使用しないでください。
- キーが画面に表示されないよう注意してください（画面共有や録画時を含む）。
- 公開・配布時は必ず HTTPS で配信してください。HTTP だとキーが盗聴される可能性があります。
- 「設定のエクスポート」を使うとキーも含まれます。共有する場合はキーを削除するか、エクスポートを使用しないでください。
- ブラウザを閉じる前にキーを消去したい場合は、設定画面で空欄にして保存するか、localStorage を手動でクリアしてください。

## 使い方（概要）
1. 右上の「API設定」を開き、使用するモデルのAPIキーを入力して保存します。
2. ゲームモードを「人間 vs 生成AI」にすると後手をAIが指します。
3. 問題が起きた場合はブラウザのコンソールにエラーメッセージが出ます。
