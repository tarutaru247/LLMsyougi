# LLMと将棋を試すデモ

このアプリはブラウザ上で動作し、利用者が用意したAPIキーを使ってLLMと対局するシンプルなデモです。

## 主な機能
- **人間 vs 生成AI** または **生成AI vs 生成AI** の対局モード
- **2つのAI思考モード**:
  - **選択式 (推奨)**: 合法手リストの中からAIに手を選ばせるモード。反則手が発生しません。
  - **生成式 (実験的)**: 盤面情報（画像・SFEN・テキスト）をAIに渡し、自力で指し手を生成させるモード。AIの推論能力を試せますが、反則手や解釈不能な回答をする可能性があります。
- **マルチモデル対応**: GPT-5.1 や Gemini など、複数のLLMモデルを切り替えて使用可能。
- **棋譜機能**: 対局の記録、一手戻し（待った）、棋譜の閲覧が可能。

## APIキー取り扱いの注意
- キーはブラウザの localStorage に平文で保存されます。共有PC・共有ブラウザでは使用しないでください。
- キーが画面に表示されないよう注意してください（画面共有や録画時を含む）。
- 公開・配布時は必ず HTTPS で配信してください。HTTP だとキーが盗聴される可能性があります。
- 「設定のエクスポート」を使うとキーも含まれます。共有する場合はキーを削除するか、エクスポートを使用しないでください。
- ブラウザを閉じる前にキーを消去したい場合は、設定画面で空欄にして保存するか、localStorage を手動でクリアしてください。

## 使い方（概要）
1. ヘッダー下の設定バーで、先手・後手のモデルを選択します。
2. 「思考モード」を選択します（通常は「選択式」推奨）。
3. 右上の「API設定」を開き、使用するモデルのAPIキーを入力して保存します。
4. ゲームモード（「人間 vs 生成AI」など）を選択し、対局を開始します。
5. 問題が起きた場合はブラウザのコンソールにエラーメッセージが出ます。
